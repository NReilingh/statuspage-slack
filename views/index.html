Statuspage-Slack Webhook Translator

Oh hi,

Statuspage webhooks are pretty cool, (and now they support case-sensitive URLs!) but the hook payloads aren't natively compatible with Slack message hooks, which usually live at an endpoint like this:

https://hooks.slack.com/services/T00000000/B00000000/XXXXXXXXXXXXXXXXXXXXXXXX

This app acts as a translation proxy for Statuspage.io webhooks. All you have to do is create a new Slack webhook, and then change the domain name to match this one.

https://statuspage-slack.glitch.me/services/T00000000/B00000000/XXXXXXXXXXXXXXXXXXXXXXXX

Here, you can even use this form to do it for you:

Once you've got the proxied URL (that points to this app), you can feed it to Statuspage's webhook subscription form, like this:

After you're signed up, you can manage your subscription as usual with the confirmation email you receive. If you want to customize the translation, feel free to remix/fork this project, and/or submit PRs to the GitHub repo!
