Oh hey,

This host is deprecated!

I decided to move the deployment to Vercel. Glitch apps run in containers that are shut down due to inactivity, and with statuspage hooks there is a risk that the container won't spin up in time for the next webhook to succeed, which will cause the webhook subscription to be paused. Using serverless functions for this makes more sense.

Go to the GitHub repo or the Vercel-hosted app!

I won't delete this, but Vercel should be more reliable and any future updates will be made there.
